This will download your private key and signed certificate in a single PKCS #12 file.

The password is 

changeit
. Despite what the password says, you should not change it, because it uses PKCS #12 encryption, which is weak. Instead you should use full disk encryption or something similar. If you're really smart you'll stick that sucker in a TPM.

To install this cert in Chrome go to 

chrome://settings/certificates
, click Import, then select the downloaded file. Enter the password 
changeit
when prompted.

This certificate expires in 1 year. Some time within the next year we will implement a way for you to exchange your cert for a new one.
